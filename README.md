# 海与船 · Sail the Mind

A small 3D world you can sail through. Four islands. Twelve people you can meet.
Press `↵` to speak with them.

A self-portrait disguised as a sailing toy — built in one afternoon as a
working prototype for the [EAZO Hackathon](https://eazo.ai/hackathon).

## Run it

```bash
python3 -m http.server 8766
# open http://localhost:8766
```

Single HTML file. Zero dependencies. Three.js from a CDN. Custom GLSL water
and sky shaders. ~77 KB of code, ~18 MB of CC0 audio.

## Controls

- `W A S D` / arrow keys — sail
- `↵` — speak with whoever you're near
- `↵` / `Esc` / click — close conversation
- Touch: left joystick to sail; tap the prompt / card to interact

## The four islands

1. **The Reckoning · 也无风雨也无晴** — 苏轼, 22 (Soul), Miguel (Coco)
2. **The Storm · 自由的代价** — 利威尔, 路明非, 皮皮鲁
3. **The Conversation · 两边怎么想** — 刘擎, Tamar Gendler, Yuval Harari
4. **The Rain Market · 雨街上的一口** — 撑伞的台南人, 椅子乐团, Claude Monet
